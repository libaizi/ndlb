# 脑洞量表在线版 — 实现计划

## 概述
构建一个纯前端 Web 应用，使用 **PeerJS (WebRTC P2P)** 实现多设备远程联机。无需自建服务器，房主创建房间获得房间码，其他人输入房间码加入，数据P2P直传。

## 技术方案
- **单文件 HTML**（内嵌 CSS + JS）
- **PeerJS**（CDN引入）：免费WebRTC信令服务，无需自建服务器
- **房主-客户端架构**：房主担任"逻辑服务器"，维护权威游戏状态，广播给所有客户端
- 原生 JS，不引入框架
- 拖拽排序：原生 HTML5 Drag & Drop
- 响应式布局，手机/电脑均可用

## 联网架构

### 通信模型
```
房主 (Host) ──── PeerJS P2P ──── 玩家A
    │                                │
    ├──── PeerJS P2P ──── 玩家B     │
    │                                │
    └──── PeerJS P2P ──── 玩家C     │
```

### 房间机制
1. 房主点击「创建房间」→ PeerJS 生成4位房间码（如 `A3F7`）
2. 其他玩家输入房间码 → 自动连接到房主
3. 房主看到所有已加入玩家列表，人齐后点「开始游戏」
4. 房间断开则游戏结束；玩家断开可重连（60秒内）

### 消息协议
房主 → 客户端（广播）:
- `state_update`: 完整游戏状态快照
- `your_number`: 某玩家的秘密数字
- `reveal_card`: 揭晓某张牌的数字
- `phase_change`: 阶段切换通知

客户端 → 房主:
- `join`: 加入房间（携带昵称）
- `peek_ack`: 已查看数字
- `sort_order`: 出题者提交排序结果
- `topic_submit`: 出题者提交题目

## 页面/状态流程

### 1. 大厅页（Lobby）
- 两个选项：「创建房间」「加入房间」
- 创建房间 → 显示4位房间码 + 等待玩家加入列表 + 「开始游戏」按钮
- 加入房间 → 输入房间码 + 输入昵称 → 进入等待

### 2. 游戏主页 — 每轮循环

#### 2a. 出题阶段
- 所有屏幕显示：当前轮次、当前出题者
- **出题者屏幕**：输入框（题目文本 + 数字1含义 + 数字10含义）+ 确认按钮
- **其他玩家屏幕**：显示「出题者正在出题，请稍候...」

#### 2b. 查看数字阶段
- 每位玩家屏幕显示自己的数字牌（仅自己可见）
  - 点击「查看我的数字」→ 数字弹出显示
  - 点击「已记住」→ 数字隐藏，标记为已查看
- 房主屏幕额外显示：所有玩家的查看状态（已查看✓ / 未查看）
- 全部查看完毕 → 自动进入下一阶段

#### 2c. 回答阶段
- 所有屏幕显示：题目 + 1/10两端提示
- 大字提示：「请各位在语音中按数字大小回答！」
- 出题者屏幕显示：「你是出题者，请仔细听大家的回答」
- 「进入排序」按钮（出题者专属）

#### 2d. 排序阶段（核心交互）
- **仅出题者屏幕**：显示所有回答者的卡片（可拖拽），从左到右=从小到大
- **其他玩家屏幕**：显示「出题者正在排序，请稍候...」
- 出题者拖拽排列完毕 → 点击「揭晓！」

#### 2e. 揭晓阶段
- **所有屏幕同步**：依次翻开数字（0.8秒间隔）
  - ✅ 递增：绿色高亮
  - ❌ 非递增：红色警告 + 独角兽🦄翻成大便💩
- 揭晓完毕 → 显示本轮结果 + 指示物状态

### 3. 轮间过渡
- 显示当前独角兽🦄 / 大便💩 数量
- 显示下一轮出题者
- 出题者点击「开始下一轮」

### 4. 终局页
- 有独角兽存活 → 🎉 全员胜利！
- 全变大便 → 💩 全员失败！
- 「再来一局」按钮

## UI 设计

### 视觉风格：简洁现代
- 白色/浅灰底色，圆角卡片
- 主色调：靛蓝 #4F46E5
- 成功色：翠绿 #10B981
- 错误色：玫红 #EF4444
- 字体：系统默认无衬线体

### 顶部状态栏（所有页面共享）
- 轮次指示：5个圆点（已完成=绿色，当前=蓝色脉冲，未到=灰色）
- 🦄 ×N | 💩 ×M
- 房间码 + 当前出题者标签
- 连接状态指示灯（绿=已连接，红=断开）

### 玩家数字卡片（查看阶段）
- 默认：头像圆 + 昵称 + 「点击查看」按钮
- 查看后：大号数字（如 "7"）+ 「已记住」按钮
- 确认后：昵称 + ✓ 标记

### 拖拽排序区（排序阶段，仅出题者）
- 横向排列的卡片
- 每张卡片：玩家昵称 + 小头像
- 拖拽时：卡片浮起+阴影
- 底部标注：← 小 | 大 →

### 揭晓动画
- 依次翻开（0.8秒/张）
- 翻开时数字放大弹出
- 正确：绿色边框 + ✅
- 错误：红色闪烁 + 💩 出现动画
